# 周一到周五 5 天健身计划

> 基于 `me.md`：178cm、70kg、无伤病、基础体能偏入门，目标为全身塑形/增肌并加入跑步；每天约 30 分钟。  
> 可用器械：坐姿推胸、坐姿飞鸟、坐姿肩上举、反握下拉、坐姿划船、器械深蹲、腿屈伸、腿弯举、哑铃等。

## 总原则

- **训练强度**：每组最后 2-3 次有吃力感，但动作不变形；不要练到完全力竭。
- **重量选择**：从能稳定完成目标次数的轻重量开始；连续两次训练都轻松完成后，下次加 1 档或加 1-2kg。
- **休息时间**：力量动作组间休息 45-75 秒；跑步与力量切换休息 1-2 分钟。
- **动作节奏**：发力 1 秒，回放 2-3 秒，保持控制，不借力猛甩。
- **热身与放松**：每天都做，热身 5 分钟，训练后拉伸 3-5 分钟。

## 每天固定流程

1. **热身 5 分钟**：快走/慢跑 3 分钟 + 肩绕环、髋绕环、徒手深蹲各 30 秒。
2. **主训练 20-22 分钟**：按当天计划完成。
3. **放松 3-5 分钟**：胸、背、腿、肩按当天训练部位拉伸。

## 周一：胸 + 肩 + 短跑适应

目标：建立上肢推力，改善胸肩线条。

| 动作 | 组数 x 次数 | 说明 |
|---|---:|---|
| 器械坐姿推胸 | 3 x 10-12 | 背贴靠垫，推起不锁死手肘 |
| 器械坐姿飞鸟 | 3 x 10-12 | 胸部发力夹合，肩不要耸 |
| 坐姿肩上举 | 2 x 10-12 | 轻重量开始，避免腰后仰 |
| 哑铃侧平举 | 2 x 12-15 | 小重量，抬到肩高即可 |
| 跑步机慢跑/快走 | 6-8 分钟 | 1 分钟慢跑 + 1 分钟快走循环 |

## 周二：背 + 二头 + 核心

目标：改善背部发力，提升体态和上肢拉力。

| 动作 | 组数 x 次数 | 说明 |
|---|---:|---|
| 反握肩距坐姿下拉 | 3 x 10-12 | 先沉肩，再把手肘向下拉 |
| 窄握坐姿划船 | 3 x 10-12 | 胸挺起，肩胛骨向后夹 |
| 哑铃弯举 | 2 x 10-12/侧 | 上臂尽量固定，不甩身体 |
| 俯身哑铃臂屈伸 | 2 x 10-12/侧 | 可练手臂后侧，轻重量控制 |
| 平板支撑 | 3 x 20-30 秒 | 腰腹收紧，不塌腰 |

## 周三：腿 + 臀 + 轻有氧

目标：强化下肢基础，提升全身消耗和稳定性。

| 动作 | 组数 x 次数 | 说明 |
|---|---:|---|
| 器械深蹲 | 3 x 10-12 | 膝盖朝脚尖方向，腰背稳定 |
| 器械腿屈伸 | 3 x 10-12 | 顶端停 1 秒，慢慢放下 |
| 器械腿弯举 | 3 x 10-12 | 感受大腿后侧发力 |
| 徒手臀桥 | 2 x 12-15 | 顶端夹臀 1 秒 |
| 坡度快走 | 6-8 分钟 | 坡度 3-6%，速度能说短句 |

## 周四：全身循环 + 跑步进阶

目标：提高心肺和全身肌肉耐力，不追求大重量。

循环做 3 轮，每个动作之间休息 20-30 秒，每轮休息 60-90 秒。

| 动作 | 每轮次数/时间 | 说明 |
|---|---:|---|
| 器械坐姿推胸 | 10 次 | 中轻重量 |
| 反握坐姿下拉 | 10 次 | 背部主动发力 |
| 器械深蹲 | 10 次 | 稳定控制 |
| 哑铃侧平举 | 12 次 | 小重量 |
| 卷腹 | 12-15 次 | 不用手拉脖子 |

循环后做 **跑步机 8 分钟**：30 秒慢跑 + 90 秒快走，循环 4 次。

## 周五：手臂 + 肩背补强 + 轻松跑

目标：补强手臂肩背，周末前不过度疲劳。

| 动作 | 组数 x 次数 | 说明 |
|---|---:|---|
| 坐姿肩上举 | 3 x 10-12 | 控制重量，肩无痛为准 |
| 哑铃侧平举 | 2 x 12-15 | 慢起慢落 |
| 窄握坐姿划船 | 3 x 10-12 | 背部夹紧，不耸肩 |
| 哑铃弯举 | 2 x 10-12 | 左右均衡 |
| 俯身哑铃臂屈伸 | 2 x 10-12/侧 | 肘部固定 |
| 轻松跑/快走 | 8-10 分钟 | 全程能讲话，不冲刺 |

## 跑步进阶规则

- 第 1-2 周：以 **跑走结合** 为主，每次跑步总时长 6-10 分钟。
- 第 3-4 周：把慢跑段从 1 分钟逐步加到 2 分钟。
- 第 5 周后：目标是连续慢跑 10-15 分钟，再考虑增加速度。
- 如果膝盖、脚踝或小腿疼痛，立即改为快走并减少跑量。

## 周末建议

- 周六：休息，或散步 30-45 分钟。
- 周日：休息，做 10 分钟拉伸和肩背放松。

## 饮食与恢复

- **蛋白质**：每天尽量吃够 100-120g 蛋白质，例如鸡蛋、牛奶、鱼虾、鸡胸、牛肉、豆制品。
- **主食**：不要完全不吃碳水；训练日前后正常吃米饭、面、土豆、燕麦等。
- **睡眠**：尽量保证 7 小时以上。
- **记录**：记录每个动作的重量、次数、体感；每 2 周拍一次正面/侧面照片观察变化。

## 安全提醒

- 任何动作出现尖锐疼痛、关节痛或头晕，应停止训练。
- 入门阶段优先动作标准，不要急着加重量。
- 如果长期胸闷、异常气短或心率不适，先咨询医生再继续提高强度。
